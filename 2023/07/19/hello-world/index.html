1.下载Node

下载地址

官方下载地址: Node.js (nodejs.org)

下载的时候记得要勾选 Add to PATH

忘了勾选Add to PATH/没添加环境变量怎么办？

如果在安装Node.js时没有勾选”Add to PATH”选项，你可以手动添加Node.js到系统的环境变量中。以下是在Windows和macOS上手动添加Node.js到环境变量的步骤：

Windows操作系统：

  1. 打开Node.js的安装目录：默认情况下，Node.js会安装在C:\Program Files\nodejs（64位系统）或C:\Program Files (x86)\nodejs（32位系统）。
  2. 复制Node.js的安装路径。
  3. 点击Windows键 + R键，打开“运行”对话框，输入sysdm.cpl并点击”确定”。
  4. 在”系统属性”对话框中，点击”高级”选项卡，然后点击”环境变量”按钮。
  5. 在”系统变量”区域下，找到名为”Path”的变量，双击进行编辑。
  6. 在”编辑环境变量”对话框中，点击”新建”按钮，然后粘贴之前复制的Node.js安装路径，并点击”确定”。
  7. 确认所有打开的对话框，使更改生效。
  8. 打开命令提示符（或重新启动命令提示符，如果它是在安装Node.js之前打开的），输入node -v和npm -v，检查Node.js和npm是否已经正确添加到环境变量。

3.安装Hexo

安装cnpm

  • 在cmd输入命令: npm install -g cnpm --registry==https://registry.npm.taobao.org

    其中

    • -g 代表全局安装
    • --registry==https://registry.npm.taobao.org 表示使用淘宝镜像安装cnpm

  • 安装成功后可以通过cnpm -v验证一下

安装Hexo

  • 命令: cnpm install -g hexo-cli

    • -g 表示全局安装
    • hexo-cli 为所安装的包

  • 安装后验证: 在 cmd 中输入命令 hexo -v 可查看 hexo 版本

4.配置GitHub

1.创建GitHub账户（已有github账户可跳过此步骤）

  • 打开GitHub官网（https://github.com/）。
  • 点击”Sign up”按钮，填写必要信息创建GitHub账户。
